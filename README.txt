How to execute:
The project was packaged using npm to encapsulate the application and dependencies that were used to create it. 

To execute the package, place the pacakge at the desired root location on your machine with node.js installed

Run the following command to unpackage the project
npm install p1-ckatnic-v1.0.0.tgz



The version number at the end of the package name may be different by the time you're reading this, so you may have to do a little bit of AGILE problem solving and respond to this radically out of date documentation. Trust the force.
